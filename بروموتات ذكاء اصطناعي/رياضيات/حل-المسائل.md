# بروموت حل المسائل الرياضية 📐

## عنوان البروموت
حل المسائل الرياضية خطوة بخطوة

## الأداة الموصى بها
**Claude** (أو ChatGPT)

## نص البروموت الجاهز
```
أنا طالب بكالوريا شعبة رياضيات. أحتاج منك حل هذا التمرين الرياضي خطوة بخطوة:

[ضع نص المسألة أو التمرين هنا]

يرجى:
1. كتابة المعطيات بشكل واضح
2. تحديد ما هو مطلوب
3. ذكر القاعدة أو النظرية المستخدمة
4. الحل خطوة بخطوة مع شرح مبسط
5. التحقق من النتيجة النهائية
```

## مثال على الاستخدام
**المدخل:**
```
أنا طالب بكالوريا شعبة رياضيات. أحتاج منك حل هذا التمرين الرياضي خطوة بخطوة:

أوجد نهاية الدالة التالية عندما تميل x إلى 2:
f(x) = (x² - 4) / (x - 2)

يرجى:
1. كتابة المعطيات بشكل واضح
2. تحديد ما هو مطلوب
3. ذكر القاعدة أو النظرية المستخدمة
4. الحل خطوة بخطوة مع شرح مبسط
5. التحقق من النتيجة النهائية
```

**المخرج المتوقع:**
سيقوم الذكاء الاصطناعي بحل المسألة مع شرح كل خطوة بالتفصيل.

## ملاحظات
- تأكد من كتابة المسألة بوضوح
- يمكنك طلب شرح إضافي لأي خطوة غير واضحة
- هذا البروموت مفيد لجميع وحدات الرياضيات (دوال، هندسة، احتمالات...)
