# بروموت شرح دروس الرياضيات 📖

## عنوان البروموت
شرح دروس الرياضيات للمبتدئين

## الأداة الموصى بها
**ChatGPT** أو **Claude**

## نص البروموت الجاهز
```
اشرح لي درس "[اسم الدرس]" من مادة الرياضيات لبكالوريا شعبة رياضيات، اشرحه كأنني مبتدئ تماماً.

يرجى اتباع التنسيق التالي:
1. تعريف بسيط للمفهوم
2. الفكرة العامة بكلمات بسيطة
3. القوانين الأساسية مع شرحها
4. أمثلة محلولة خطوة بخطوة
5. تمارين للتدريب مع الحلول
6. نصائح لتجنب الأخطاء الشائعة

استخدم لغة عربية بسيطة ومباشرة.
```
