# بروموتات الفيزياء

## 1. شرح الدروس

**الأداة الموصى بها:** Claude أو ChatGPT

**نص البروموت:**
```
اشرح لي درس "[اسم الدرس]" من الفيزياء للبكالوريا شعبة [رياضيات/تقني رياضي/علوم].

يرجى:
1. المفهوم الأساسي
2. القوانين المهمة
3. أمثلة تطبيقية
4. وحدات القياس
5. نصائح للتميز
```

---

## 2. حل المسائل

**الأداة الموصى بها:** Claude

**نص البروموت:**
```
احل هذا التمرين مع شرح كل步骤:
[نص التمرين]

المعطيات:
[المعطيات]
المطلوب:
[المطلوب]
```

---

## 3. أسئلة تدريبية

**نص البروموت:**
```
أنشئ 10 أسئلة تدريبية في [اسم الموضوع] للفيزياء.
```


## 4. تجارب مخبرية

**نص البروموت:**
```
اشرح تجربة [اسم التجربة] في الفيزياء:
- الهدف
- الأدوات
- الخطوات
- استخراج النتائج
```
