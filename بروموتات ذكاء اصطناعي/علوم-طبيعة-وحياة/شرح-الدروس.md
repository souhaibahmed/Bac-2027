# بروموتات علوم الطبيعة والحياة

## 1. شرح الدروس

**نص البروموت:**
```
اشرح لي درس "[اسم الدرس]" في علوم الطبيعة والحياة للبكالوريا.
```

---

## 2. رسم بيولوجي

**نص البروموت:**
```
ارسم واشرح [اسم الرسم]. مثلاً: تركيب الخلية - دورة الماء - السلسلة الغذائية
```

---

## 3. أسئلة تدريبية

**نص البروموت:**
```
أنشئ أسئلة تدريبية في [اسم الموضوع] لعلوم الطبيعة.
```
