## 🚀 كيفية الاستخدام

1. **اختر المادة المطلوبة** من المجلدات أعلاه
2. **افتح ملف الشرح** المطلوب
3. **انسخ نص البروموت** 
4. **الصقه في أداة الذكاء الاصطناعي**
5. **خصص البروموت** بمعلوماتك

## 💡 نصائح للاستخدام الفعال

- كن محدداً في طلبك (اذكر اسم الدرس، الشعبة، الوحدة)
- اطلب تكرار الشرح إذا لم تفهم
- استخدم أمثلة تطبيقية
- لا تتردد في طرح أسئلة متابعة
