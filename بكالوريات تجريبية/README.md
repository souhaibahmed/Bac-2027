## 📝 نظام تسمية الملفات

تم اعتماد نظام التسمية التالي:
```
{السنة}-{المؤسسة}-{المادة}.pdf
```

### أمثلة:
- `2026-ثانوية-الشهداء-رياضيات.pdf`
- `2025-ديوان-المدارس-فيزياء.pdf`
- `2026-مؤسسة-النجاح-عربية.pdf`

## ✅ معايير الجودة

- ✅ صيغة PDF فقط
- ✅ جودة صورة واضحة وقابلة للقراءة
- ✅ الملفات مسماة بشكل صحيح وفق نظام التسمية
- ❌ لا ملفات مكررة
- ❌ لا ملفات تالفة
