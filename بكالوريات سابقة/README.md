## 📝 نظام تسمية الملفات

تم اعتماد نظام التسمية التالي:
```
{السنة}-{الشعبة}-{المادة}.pdf
```

### أمثلة:
- `2024-رياضيات-فيزياء.pdf`
- `2023-آداب-عربية.pdf`
- `2022-علوم-تجريبية-علوم.pdf`

## ✅ معايير الجودة

- ✅ صيغة PDF فقط
- ✅ جودة صورة واضحة وقابلة للقراءة
- ✅ الملفات مسماة بشكل صحيح
- ❌ لا ملفات مكررة
- ❌ لا ملفات تالفة
