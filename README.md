## 📋 عن المشروع

**Bac-2027** هو مستودع GitHub تعليمي شامل، يهدف إلى مساعدة طلاب البكالوريا في الجزائر على التحضير الفعّال لامتحانات البكالوريا لعام 2027. يجمع المستودع في مكان واحد جميع الموارد التعليمية الرقمية المفيدة.

### ✨ ما يحتويه المستودع:

- 📺 **قنوات يوتيوب** تعليمية لكل مادة
- 🌐 **مواقع إلكترونية** مفيدة وملخصات
- 🤖 **أدوات ذكاء اصطناعي** لتعزيز التعلم
- 💡 **بروموتات ذكاء اصطناعي** جاهزة للاستخدام
- 📚 **بكالوريات سابقة** (آخر 5 سنوات)
- 📝 **بكالوريات تجريبية** من المدارس
- 📒 **ملخصات** الدروس بصيغة PDF
