## 💡 كيف تستفيد من هذا المستودع؟

- اختر المادة التي تريد دراستها
- ستجد قنوات يوتيوب مع الروابط المباشر

### 2. المواقع والتطبيقات
-تصفح مجلد `Websites/` و `Apps/`
- ستجد أفضل المنصات التعليمية الجزائرية

### 3. الذكاء الاصطناعي
- استخدم الأوامر في مجلد `AI_Prompts/` مع ChatGPT أو Claude
- ستساعدك في شرح الدروس وإنشاءاختبارات والتدرب
