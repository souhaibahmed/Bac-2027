## 📝 نظام تسمية الملفات

تم اعتماد نظام التسمية التالي:
```
{الوحدة}-{عنوان-الدرس}.pdf
```

### أمثلة:
- `الوحدة1-الدوال-الحسابية.pdf`
- `الوحدة3-الكهرباء.pdf`
- `الوحدة2-الخلية.pdf`

## ✅ معايير الجودة

- ✅ صيغة PDF (مفضل) أو Markdown
- ✅ جودة واضحة وقابلة للقراءة
- ✅ الملفات مسماة بشكل صحيح وفق نظام التسمية
- ❌ لا ملفات مكررة
- ❌ لا ملفات تالفة أو غير مكتملة
